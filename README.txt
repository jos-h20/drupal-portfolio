Shift Cipher Module
Allows users to enter a phrase and have it encrypted, 4.29.2016

By Josh Overly

Description

Users can enter a phrase, determine the number of spaces to shift, determine which direction they want the shift to occur and control how their phrase is encrypted.

Setup/Installation Requirements

You will need to have MAMP or WAMP installed.

Go to https://github.com/jos-h20/drupal-portfolio.
Copy the directory.
From your terminal, change your directory to desktop.
Type in git clone https://github.com/jos-h20/drupal_portfolio.git
Open MAMP and click on Preferences
Click on the folder icon and select drupal_portfolio
Start servers
Open WebStart Page
Select Tools/PHPMyAdmin
Click on Import
Choose file/go to drupal_portfolio/sites/db-backup
Click on the zipped database
Click Go
Once database has downloaded, click josh_database
Go to Privileges, click Add User
For user name put in josh
change host to local
For the password, type in povagabond4u
Click Go
In your browser, type in localhost:8888
Select standard installation/select English
Enter josh_bookstore_database, user name: josho, password: povagabond4u
Under advanced options, change host to 127.0.0.1
And port to 8889
The user name and password are the same for the site.
